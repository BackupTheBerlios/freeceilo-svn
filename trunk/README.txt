Free Ceilo is an interface for ceilometer CL-31.

It contains to programs freeceilo and fcgetdata.

fcgetdata --> Read data from the ceilometer and store the data in the FreeCeilo directory inside the home folder.(reading data via serial port either ttys0 ,ttys1 or ttys2)

freeceilo --> Plots the data.To plot the data you have to choose the data file (.dat files only) 		      by clicking the filemenu and choose open.
